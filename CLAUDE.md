# GAD MIVNIM — אתר תדמיתי

אתר תדמיתי בעברית (RTL) לקבוצת בנייה "GAD MIVNIM" (גד מבנים), הרצליה, ישראל.
הסלוגן: **"בכל מקום במפה"** + **"משנים את הדרך לבנייה נכונה!"**

## טכנולוגיה
- HTML + CSS + JavaScript בלבד (בלי frameworks, בלי build tools).
- קובץ ראשי: `index.html`, עיצוב ב-`css/style.css`, סקריפטים ב-`js/script.js`.
- כל הדפים בעברית, `dir="rtl"`, פונט Heebo או Assistant מ-Google Fonts.
- רספונסיבי מלא — מובייל ודסקטופ.
- תמונות אמיתיות של החברה נמצאות בתיקיית `images/` (image1.jpeg ... image107.png). השתמש בהן בכל האתר: hero, כרטיסי שירותים, גלריה, פרויקטים.

## סגנון עיצובי
- צבע מותג ראשי: ירוק (למותג יש "משמעות לצבע הירוק"). משני: שחור/אפור כהה + לבן.
- מראה יוקרתי ומקצועי של חברת בנייה: תמונות גדולות, טיפוגרפיה נקייה, הרבה אוויר.
- אווירה משפחתית וחמה בטקסטים — "אווירת שירות משפחתית ואישית לכל לקוח".

## מבנה האתר (One-page עם עוגנים, או דפים נפרדים — עדיף one-page בשלב ראשון)

לוגו GAD MIVNIM + קישורים: דף הבית, אודות הקבוצה, השירותים שלנו, הפרויקטים שלנו, השותפים שלנו, הצוותים שלנו, גלריה, צור קשר.
בורר שפה (עברית / English / Russian) — בשלב ראשון רק עברית פעילה.

### 2. Hero
תמונת בנייה גדולה, כותרת "GAD MIVNIM — בכל מקום במפה", תת-כותרת "משנים את הדרך לבנייה נכונה!", כפתור CTA "צרו קשר".
שורת משנה: "לרשותכם אולם תצוגה לתצוגות בנייה ועיצובים | הרצליה, ישראל".

### 3. אודות הקבוצה — "מסורת משפחתית"
- מייסד הקבוצה והשותפים למלאכה.
- "מדוע נולד הרעיון לאחד את הבנייה והעיצובים לבית אחד ולקבוצה שמנצחת על הכל!"
- ערכי יסוד: איכות, מקצועיות, יחס משפחתי ואישי. עבודה שיתופית מלאה.
- "בחירתנו היא שותפות מלאה — לקבלת תוצאות שלא פחות ממצוינות".

### 4. השירותים שלנו — 5 חטיבות (כרטיסיות עם תמונה + כותרת + משפט)
**חטיבת תכנון והנדסת בנייה** — "תכנון נכון, חיסכון מוכח":

**חטיבת בינוי ותשתיות** — "בונים אתכם מבנה נכון":
בנייה והרחבות מבנים ("מבנה חדש נולד!"), שיקום שחזור ושימור מבנים, שיפוצים והשבחת נכסים ("נצייר לכם ציור חדש!"), ליקויי בנייה ("נתקן את זה!"), תשתיות ופיתוח ("הכל מתחיל ביסודות טובים"), אספלט וכבישים.

**חטיבת שותפויות נדל"ן** — "עושים שותפות מתוך כבוד הדדי וערכים משפחתיים":
תמ"א 38 ("נעשה את זה ביחד ובהצלחה גדולה"), פינוי בינוי ("תמיד כיף לקבל חדש!").

**חטיבת השירותים שלנו** — "שירות מכל הלב":
מערכות בידוד ואיטום, עבודות בגבהים, ניקוי חול והתזות, גגות רעפים ופרגולות, גננות ופיתוח סביבתי, גבס ודקורציה, צבע וציפויים, אינסטלציה וביוב, חשמל ותקשורת.

**חטיבת המיוחדים שלנו** — "מובילים שינוי למבנה שלכם!":
ניהול ואחזקת מבני מגורים ומסחר, תיווך נכסים.

מטבחים ("המטבח זה הלב של הבית"), נגרות בהתאמה אישית, רהיטים מעוצבים, ריצוף וחיפויים, משטחים, פרקטים, כלים סניטריים, ריהוט, תאורה, דלתות פנים, דלתות כניסה, שערים וגדרות, זגגות ומסגרות בנייה.

(בשלב ראשון הטופס רק מציג הודעת "קוד המשתמש בדרך אליכם.. תודה שבחרתם לצפות באתר הבית שלנו" — בלי שרת.)
